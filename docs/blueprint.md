# **App Name**: YouthMind

## Core Features:

- Mood Detection and Response: Detect user's mood from text input and provide empathetic, supportive responses using a generative AI tool.
- Personalized Recommendations: Suggest coping tips, mindfulness exercises, or creative prompts tailored to the user's detected mood using a generative AI tool.
- Multilingual Support: Offer support and content in multiple languages based on user preference. Limited to supported languages
- Journaling Prompts: Provide creative journaling and art prompts using a generative AI tool.
- Peer Support Threads: Enable anonymous peer-support threads with AI moderation.
- Crisis Escalation: Escalate crisis cases to emergency helplines with clear, immediate action upon detecting specific keywords.
- UI for Mood Input: Allow users to input their current mood via text.

## Style Guidelines:

- Primary color: Soft lavender (#D1C4E9) to evoke calmness and serenity.
- Background color: Very light lavender (#F3E5F5) to create a gentle and soothing atmosphere.
- Accent color: Pale blue (#BBDEFB) to provide a sense of peace and clarity.
- Body and headline font: 'PT Sans' (sans-serif) for a modern yet approachable feel, suitable for both headlines and body text.
- Use simple, clean icons to represent different moods and activities.
- Maintain a clean and minimalist layout to reduce visual clutter and promote ease of use.
- Incorporate subtle animations, such as gentle transitions and fades, to enhance the user experience.
